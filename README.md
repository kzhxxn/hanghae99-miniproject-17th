천만 반려인 시대!
펫심을 겨냥하여 반려동물과 함께할수 있는 캠핑장 추천뿐만 아니라,
테마별 해시태그, 별점항목을 기록하고 공유할수 있는 펫심저격 캠핑 아카이브입니다.

## 기능 및 API 설계

- 회원가입과 jwt 토큰을 사용한 로그인, 로그아웃
- selenium 를 이용한 캠핑 목록 크롤링
- 캠핑장 별로 리뷰 평점, 리뷰 코멘트 작성 기능
- 캠핑장 리스트 도시별, 태그별, 조회수별 sorting!
- 무한 슬라이드 배너
